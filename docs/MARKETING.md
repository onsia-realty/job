# 온시아 Job 마케팅 전략

## 목표
**"부동산 = 돈 벌 수 있다"는 인식 전환 → 젊은 층 유입 → 자연스러운 이력서 등록/지원**

## 현실 인식
- 부동산 = "아저씨 직업", "힘들다", "사기꾼" 이미지
- MZ세대는 부동산 중개업을 커리어로 고려 안함
- 자격증 없는 사람은 접근조차 안함
- 공인중개사 34개월 연속 폐업 > 개업 (월 -376명)

## 전략: "부동산 업계 진입 장벽 낮추기 + 바이럴"

---

## 아이디어 리스트

### 1. "면접비 지원" 캠페인
```
"면접만 봐도 5만원!"
- 면접 인증샷 올리면 커피 기프티콘
- 구인 업체가 면접비 부담 (프리미엄 공고 옵션)
- SNS 바이럴: "부동산 면접 보고 5만원 받음 ㅋㅋㅋ"
- 채널: 트위터, 스레드, 인스타
```

### 2. "부동산 연봉 계산기" (바이럴 콘텐츠)
```
"너의 부동산 예상 연봉은?"
- 간단 퀴즈: 성격/지역/관심사 5개 질문
- 결과: "서울 강남 상가 전문가 → 예상 연봉 8,000만원"
- 결과 공유 버튼 (인스타 스토리/트위터)
- 공유하면 → 이력서 등록 유도 팝업
```

### 3. "0원 부동산 도전기" 시리즈 (AI 웹툰/영상)
```
"자격증 없이 부동산에서 첫 월급 받기까지"
- AI 웹툰: 20대가 부동산 사무실 취업 → 3개월 후 300만원
- 유튜브 숏츠/릴스: "부동산 신입 브이로그" 컨셉
- 마지막에 "온시아에서 이력서 등록하고 시작하세요"
```

### 4. "무자격 OK" 공고 필터
```
현재: 모든 공고가 "경력 N년 이상"
개선: "자격증 없어도 OK" / "신입 환영" 필터 추가
→ "나도 할 수 있네?" 심리적 장벽 제거
→ 공고 클릭 → 이력서 등록 유도
```

### 5. "부동산 인턴십" 프로그램
```
"2주 체험 → 맞으면 정규직"
- 구인 업체와 제휴: 단기 체험 프로그램 운영
- 체험 후 이력서 자동 등록
- SNS: "부동산 2주 알바 후기" 콘텐츠 생산
```

### 6. "친구 추천하면 N만원" 리퍼럴
```
- 이력서 등록한 사람이 친구 초대
- 친구도 이력서 등록하면 → 둘 다 기프티콘
- 바이럴 루프 생성
```

### 7. "이력서 등록만 해도 추첨"
```
매주 금요일 추첨
- 1등: 에어팟
- 2등: 배민 5만원
- 3등: 스벅 (10명)
→ 이력서 등록 = 응모권
```
